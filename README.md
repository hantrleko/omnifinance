# OmniFinance 全能理财家

[![CI](https://github.com/hantrleko/omnifinance/actions/workflows/ci.yml/badge.svg)](https://github.com/hantrleko/omnifinance/actions/workflows/ci.yml)

当前版本：`v2.2.0`

OmniFinance 是一个基于 **Streamlit** 的个人财务规划与轻量投资分析平台。项目将预算、储蓄、债务、保险、退休、投资组合优化、策略回测、压力测试和行动计划整合到一个多页面应用中，目标不是替代专业投顾，而是帮助用户完成：

> 财务状态诊断 → 风险与机会识别 → 情景模拟 → 优先行动计划 → 报告导出与复盘。

---

## 项目定位

OmniFinance 适合作为：

- 个人财务规划工具箱；
- 金融数据分析与 Streamlit 产品化练习项目；
- 投资组合优化、回测、退休规划等金融计算模块的开源样例；
- 面向作品集展示的端到端金融应用。

本项目目前定位为 **学习、研究与个人决策辅助工具**。所有结果均依赖输入假设、历史数据和模型参数，不构成投资、保险、税务或法律建议。

---

## 核心工作流

```text
1. 输入个人财务资料
   ↓
2. 生成财务健康评分
   ↓
3. 识别储蓄、债务、保险、投资、退休等关键机会
   ↓
4. 使用压力测试与 what-if 模拟比较不同方案
   ↓
5. 形成 90 天行动计划
   ↓
6. 导出 Markdown / HTML 报告，用于复盘与迭代
```

## 功能模块

### 1. 平台概览

- 🏠 仪表盘首页
- 🧠 个人财务决策简报
- 🛰️ 财务机会雷达
- 🧯 财务压力测试实验室
- 🧪 行动影响模拟器
- 📤 Markdown / HTML 报告导出

### 2. 基础理财管理

- 💰 复利计算器：长期复利、通胀影响、收益率敏感性
- 🎯 储蓄目标计算器：目标金额、月存额、达成时间反推
- 💡 预算分配建议器：50/30/20 预算框架与储蓄率建议
- 🏫 教育基金规划器：教育费用、通胀、资金缺口测算

### 3. 资产与债务管理

- 🏠 资产净值追踪器：资产、负债与净资产趋势
- 🏦 贷款计算器：等额本息、提前还款、总利息比较
- 🛡️ 保险产品测算器：保费效率、现金价值、IRR、通胀折现
- 💳 债务还清规划器：雪球法与雪崩法还款路径
- 🏘️ 房产投资分析器：租金收益、现金流和投资回报

### 4. 投资分析引擎

- 📊 实时报价面板：基于 yfinance / akshare 的行情查询
- 📐 投资组合优化器：均值-方差优化、最大夏普、最小方差、有效前沿
- 📈 策略回测器：MA、RSI、MACD、布林带策略，支持手续费与滑点
- ⚖️ 资产再平衡模拟器：定期再平衡、阈值再平衡、交易影响
- 💱 外汇对冲计算器：汇率风险、对冲成本与敞口管理

### 5. 高级人生规划

- 🏖️ 退休金估算器：退休缺口、月存需求与敏感度分析
- 🎲 蒙特卡洛模拟：目标达成概率与风险分布
- 🏦 税务优化提款策略：退休提款顺序与资金寿命比较
- 📜 历史回测储蓄模拟：基于历史市场数据验证投资路径
- 📔 财务日记：记录财务决策、复盘和情绪标签

### 6. 分析与工具

- 🧾 税务计算器
- 🔬 场景对比分析器
- 📅 财务日历
- 🔔 财务提醒管理
- 💱 货币转换器
- 🧮 科学金融计算器
- 🔎 股票筛选器
- 📒 收支记账本

## 快速开始

请在仓库根目录执行：

```bash
pip install -r requirements.txt
streamlit run app.py
```

开发环境建议安装：

```bash
pip install -r requirements-dev.txt
```

### 1 分钟体验路径

如果你是第一次接触 OmniFinance，推荐按这个最短路径开始：

1. 运行 `streamlit run app.py`。
2. 在左侧进入 **🧭 决策中枢**，按“推荐初始化路径”完成：
   - 🏠 预算分配建议器（建立月收入/支出与可储蓄基线）
   - 🏠 资产净值追踪器（建立净资产与负债基线）
   - 🏖️ 退休金估算器（估算退休缺口与月补充需求）
3. 回到左侧或顶部直接进入 **🏠 仪表盘首页**，查看：
   - 财务健康评分、机会雷达、压力测试；
   - 机会清单与 90 天行动建议；
   - 自动导出决策简报。

### 首页入口与常见场景对应

| 入口 | 主要用途 | 先决条件 |
|---|---|---|
| 决策中枢 | 串联诊断、机会、压力测试、行动模拟 | 无（先完成关键输入效果更好） |
| 仪表盘首页 | 查看全部页数据汇总与可执行建议 | 建议先填预算/净资产/退休 |
| 预算分配建议器 | 生成可持续月现金流与储蓄率 | 收入支出数据 |
| 储蓄目标计算器 | 反推目标金额所需时间与月存额 | 目标金额与可用期限 |
| 退休金估算器 | 估算退休缺口与补充策略 | 收入增长率与退休目标 |
| 税务计算器 | 计算应纳税额、税后月入 | 工资/商业所得相关参数 |

### 关键指标口径（首页展示卡片）

- **月结余（现金流）**：每月可用于储蓄/投资的金额，低于 0 代表赤字压力较大。  
- **退休缺口**：目标退休资产与当前可用资产/预计储蓄轨道的差额；小于等于 0 代表目标可达成。  
- **健康评分**：基于预算、净资产、税务、保险和退休模块综合打分，包含可执行改进建议。  
- **机会雷达**：对可立即改进的储蓄、债务、保障、投资与退休子项按影响力进行排序。  
- **压力测试**：模拟收入下降、支出上浮、资产回撤下的现金流安全垫（单位月）并输出应对行动。
